MEGpro: CHP Calculator 
=-=-=-=-=-=-=-=-=-=-=
Version: 1.2.0

What is it?
-----------
A powerful time saving app that allows users to create a generator that best suits their needs
and makes it easy to run heat requirement calculations without confusion.
CHP = "Combined Heat & Power"
